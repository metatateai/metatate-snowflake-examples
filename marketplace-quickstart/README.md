# Marketplace Quick Start

Use this folder when preparing Snowflake Provider Studio quick-start examples for the Metatate listing.

The reliable Phase 1 path is the SQL quick-start. Listing notebook attachment is not a blocker for this phase because Provider Studio notebook attachment is currently affected by a Snowflake-side issue.

## Files

- `provider-studio-sql-example.md`: fields to paste into Provider Studio's Add SQL example tab

## Goal

Show a buyer that Metatate can make Snowflake data agent-ready by exposing:

- governed table discovery
- table-level decision context
- column-level meaning
- agent-ready wrapper functions
